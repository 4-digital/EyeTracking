# EyeTracking
O interfata Hardware-Firmware-Software pentru a actiona un carut de catre persoane cu invaliditati
 ## Modalitati de interactiune:
 * miscarile ochilor
 * Clipitul
 * Miscarea maxilarului pe verticala
 * Miscarea capului este posibila doar cu limitari
 * Posibilitatea de a adauga si alti senzori pentru a se potrivi mai multor scenarii. ex flux de aer, etc.

 ## Modul Comanda
   * posibilitatea de a actiona direct joystick-ul carutului fara a-l demonta
   * optiunea de a actiona driverele motoarelor prin PWM
   * Modalitate de a atentiona insotitorul prin semnale luminoase/acustice
   * Vizualizarea functiilor principale prin LED-uri sau LCD
   * mesaje audio in casca sau mic difuzor
